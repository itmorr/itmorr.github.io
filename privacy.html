Moneypi YouTube Privacy Policy

Last Updated: September 1, 2025

Moneypi YouTube is a personal project running on a Raspberry Pi to automatically schedule YouTube live streams when motion is detected via a webcam. This app uses the YouTube Data API (scope: https://www.googleapis.com/auth/youtube) to create and manage live stream events.

Data Usage:

  • The app accesses your YouTube account to schedule and manage live streams.
  • Data is processed locally on the Raspberry Pi and not shared with any third parties.
  • No user data is stored beyond what is necessary to schedule streams, and all data remains on the device.
